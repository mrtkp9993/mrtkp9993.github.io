Stock Return Analysis

This project aims to simplify the analysis of financial instruments' past performances with just a few clicks. By leveraging advanced data analytics and visualization techniques, we have developed an intuitive platform that allows users to easily access and interpret historical data of various financial instruments. This tool lets users quickly assess the performance trends, volatility, and key indicators of different assets, such as stocks, bonds, or commodities. By providing a user-friendly interface and comprehensive visualizations, we empower individuals and professionals to make informed investment decisions, identify patterns, and gain valuable insights into the historical performance of financial instruments. Our project streamlines the analysis process, saving time and effort while enhancing the understanding and evaluation of investment opportunities.

Techniques used: Deep Learning, Image Processing
Technologies used: Python, Streamlit, StatsModels, arch

Pre-Covid and Post-Covid Twitter sentiment analysis

This project focuses on the analysis of Twitter sentiment in both the pre-Covid and post-Covid eras. By leveraging advanced natural language processing techniques, we delve into the vast amount of Twitter data to uncover valuable insights about public sentiment and its evolution over time. Through this analysis, we aim to understand the shifts in attitudes, emotions, and opinions expressed on social media before and after the Covid-19 pandemic. By shedding light on these sentiments, we gain a deeper understanding of the social and emotional impact of the global crisis, paving the way for informed decision-making and proactive responses in the future.

Techniques used: Supervised Learning, Contingency tables, TF-IDF, SMOTE
Technologies used: Python, NLTK, scikit-learn, imbalanced-learn, spaCy

Image Forgery Localization with Deep Learning

This focuses on image forgery localization using deep learning techniques and the generation of masks based on the predicted results. With the rise of digital manipulation and the proliferation of fake images, it has become crucial to develop robust methods for detecting and localizing forged regions within images. By leveraging the power of deep learning algorithms, we have trained a model to accurately identify areas that have been tampered with. Furthermore, we have developed a mechanism to generate masks that precisely outline the forged regions, visually representing the detected manipulations. Our project aims to contribute to the fight against image forgery by providing an efficient and reliable tool for forensic analysis and verification of authenticity.

Techniques used: Deep Learning, Image Processing
Technologies used: Python, PyTorch, Gradio, OpenCV

Multi-class weather recognition from still image with Deep Learning

This project focuses on the development of a deep-learning model for multi-class weather recognition from still images. By utilizing advanced convolutional neural networks (CNNs) and image processing techniques, we have created an accurate and efficient system capable of categorizing images into various weather conditions. This technology has numerous applications, ranging from automated weather monitoring to enhancing image search and retrieval systems. Our project aims to provide a valuable tool for weather analysis and enable more informed decision-making in various industries that rely on weather information.

Techniques used: Deep Learning, Image Processing
Technologies used: Python, PyTorch, OpenCV
